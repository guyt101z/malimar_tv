== README

This application was built for a client for their video streaming service. It includes a 
CMS for their content, a billing system, a basic CRM/User management tool, and migration
tool from their old MS Access database.

=== Technology
* Rails 4.1
* MySQL
* AWS Load balancer
* Redis/Resque
* ActiveMerchant with PayPal

==== This repo is for demo purposes only.
